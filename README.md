<h2>프로젝트 : 쉐어하우스&하우스메이트 매칭 웹서비스</h2>
<p>* 목적 : 공용 주거공간에서 거주하고자하는 사람과 거주자가 필요한 쉐어하우스 간 매칭 플랫폼</p>
<p>* 개발 기간 : 2021.04.15 ~ 2021.05.25</p>
<p>* 기능설명 : </p>
<p>
  
- index 쉐어하우스/하우스메이트 최신목록 출력 <br>
  
- 로그인, 로그아웃 (유효성검사) <br>

- 카카오 지도 API , 각 지역의 모집중인 하우스메이트의 클러스터러 <br>

- 모집중인 쉐어하우스의 마커 및 오버레이 <br>

- Node.js 인스턴스 웹 메신저 서버구축 (실시간 채팅, DB연동, 채팅방 나가기, 신고하기), <br>

- 하우스메이트 목록 (페이징, 검색필터 기능 지원) <br>

- LOGO, 웹 디자인, UI설계, 공통 CSS 배포
</p><br>

<p>* 개발환경 : </p>
<p>
  
- FRONT <br>
EJS, HTML5 , CSS3, JavaScript, jQuery <br>
<br>
  
- BACK-END <br>
JAVA, JSP, Spring, My Batis <br>
 <br>
  
- DB / SERVER <br>
Oracle, Node.js, GitHub, Tomcat9 <br>
 <br>
  
- TOOL <br>
Eclipse, Spring, Visual Studio Code, Notion, Oven <br>
</p>
